卷积神经网络

Posted on 2019-01-31

卷积神经网络（CNN）的定义

一般地，我们可以将CNN视为人工神经网络，它是具有专门可以检测或挑选pattern的一些结构。
这种pattern detection使CNN对图像分析非常有用。

如果CNN只是一个人工神经网络，那么它与标准多层感知器或MLP的区别是什么？
它们的区别在于：CNN有隐藏的层，称为卷积层。

卷积层

与任何其他层一样，卷积层接收输入，以某种方式转换输入，然后将转换后的输入输出到下一层。
卷积层的输入称为输入通道，输出称为输出通道。
对于卷积层，发生的变换称为卷积运算。卷积运算是深度学习社区使用的术语。在数学上，卷积层执行的卷积运算实际上称为互相关。

过滤器和卷积运算

如前面所提到的那样，卷积神经网络能够检测图像中的特征（detect pattern）。
对于每个卷积层，我们需要指定图层应具有的过滤器数量，作为检测特征所用。

Patterns

patterns具体是指什么呢？
当我们说过滤器能够检测patterns时，想想在任何一张图片中我们需要获得怎样的信息才能使它被检测出来呢？
边缘，形状，纹理，对象等。这些是我们所说的patterns。

  • 边缘
  • 形状
  • 纹理
  • 曲线
  • 对象
  • 颜色

过滤器在图像中可以检测到的一种模式是边缘，因此该滤波器将被称为边缘检测器。除了边缘，一些过滤器可能会检测角。有些可能检测圆圈，有些检测矩形。这些简单的几何滤波器就是我们在卷积神经网络开始时所看到的。

网络越深，过滤器就越复杂。在后面的图层中，我们的滤波器可以检测特定的物体，如眼睛，耳朵，头发或毛发，羽毛，鳞片和喙，而不是边缘和简单的形状。

在更深的层中，过滤器能够检测更复杂的物体，如犬，猫，蜥蜴和鸟类。

例子

下面我们来具体看一个手写数字识别的例子

假设我们有一个卷积神经网络接受手写数字的图像（比如来自MNIST数据集），我们的网络正在将它们分类为各自的类别，即1,2,3等

过滤器 (pattern detectors)

我们假设网络的第一个隐层是卷积层，正如前面所提到的那样，当添加一个卷积层到我们的模型的时候，我们必须指定过滤器的数量。过滤器的数量决定了输出通道的数量。

从数学上讲，滤波器可以被认为是一个相对较小的矩阵（张量），为此，我们决定该矩阵具有的行数和列数，并且该矩阵内的值用随机数初始化。

对于我们的第一个卷积层，我们将指定我们希望该层包含一个大小为3 x 3的过滤器。

卷积层

让我们看一下卷积操作的示例动画：

此动画展示了没有数字的卷积过程。我们在底部有一个蓝色的输入通道。底部阴影的卷积滤镜在输入通道上滑动，绿色为输出通道。

  • 蓝色（底部） - 输入通道
  • 阴影（蓝色顶部） - 3 x 3卷积滤波器
  • 绿色（顶部） - 输出通道

对于蓝色输入通道上的每个位置，3 x 3滤波器进行计算，将蓝色输入通道的阴影部分映射到绿色输出通道的相应阴影部分。该卷积层接收输入通道，并且滤波器将在输入自身的每个3×3像素集上滑动，直到它在整个图像的每3×3像素块上滑动。

卷积操作

这种滑动被称为卷积，所以实际上，我们应该说这个滤波器将在输入的每个3 x 3像素块上进行卷积。

蓝色输入通道是来自MNIST数据集的图像的矩阵表示，该矩阵中的值是图像中的各个像素。这些图像是灰度图像，因此我们只有一个输入通道。

  • 灰度图像具有单个颜色通道
  • RGB图像具有三个颜色通道

此输入将传递到卷积层。

正如刚才所讨论的那样，我们已经指定第一个卷积层只有一个滤波器，这个滤波器将在输入的每个3 x 3像素块上进行卷积。当滤波器落在其第一个3×3像素块上时，将计算并存储滤波器本身与来自输入的3×3像素块的点积。以此遍历。

例如，我们将滤波器的点积与第一个3 x 3像素块相对应，然后将结果存储在输出通道中。然后，过滤器滑动到下一个3 x 3块，计算点积，并将值存储为输出通道中的下一个像素。

在此过滤器对整个输入进行卷积之后，我们将留下输入的新表示，现在将其存储在输出通道中。此输出通道称为特征图（feature map）。 这个绿色输出通道成为下一层作为输入的输入通道，然后我们刚刚通过滤波器进行的这个过程将发生在具有下一层滤波器的新输出通道上。

令人惊讶的是，模式检测器是由网络自动导出的。过滤器值以随机值初始化，它的值随着网络在训练期间的学习而变化。Pattern detectors(模式检测器)随着网络的学习而出现。

过去，计算机视觉专家会手动开发滤波器（模式检测器）。其中一个例子是Sobel滤波器，一种边缘检测器。但是，通过深度学习，我们可以使用神经网络自动学习这些过滤器！
